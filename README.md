CloudKid Captions
================

API for controlling the playback of captions in CloudKidOS

##Documentation##

http://cloudkidstudio.github.io/CloudKidCaptions/

##Installation##

CloudKid Captions can be install using Bower.

```bash
bower install cloudkid-captions
```
